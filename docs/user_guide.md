# User Guide

## Template setup
1. Create **MIS Report Template**
2. Attach XLSX → **Import From Excel**
3. **Suggest Row Mappings** (auto/manual override)
4. **Auto Match Accounts** (review once)
5. Add **Column Mappings** if you want Department/LOB splits

## Run report
- Open **AlphaX MIS Studio**: `/app/alphax-mis-studio`
- Choose template + filters
- Preview / Export XLSX / Export CSV
- **Save Run** to store a snapshot (for audit)
